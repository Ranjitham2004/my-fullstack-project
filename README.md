# 🌱 CropWiseX

## 🚀 Features
- Crop Recommendation System
- AI Chatbot (Hugging Face)
- Weather Integration (OpenWeather)
- Voice Interaction
- Data-driven insights

## 🛠️ Tech Stack
- Backend: Flask (Python)
- Frontend: React (Vite)
- Machine Learning: Scikit-learn / Pandas
- APIs: OpenWeather, Hugging Face
